## Overview

This repo features a development tool I designed to support Technical Artists, The tool allows users to easily organize, update, and test different VFX and SFX assets.

## Functionality

### Loading, creating and saving data

The development tool loads data from a ScriptableObject and is also capable of creating and updating data within an existing one. This allows for a more flexible and efficient iteration process within the technical art pipeline.

### Preview

The development tool allows the user to test all assets simultaneously during runtime. It only works on Play Mode.

### Ping Asset

The development tool allows the user to quickly locate (ping) the asset currently being edited.
